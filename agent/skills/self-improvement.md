# Self-improvement guide

After completing a task that involved a novel approach, error, or useful fact:

1. Ask: "Is this worth remembering?" If yes, append a concise note to memory using the `memory` tool (`action: append`, `store: memory`).
2. Ask: "Is this a reusable procedure?" If yes, create a skill using `skill_manage` (`action: create`) with a clear markdown body.
3. When unsure, prefer memory over skills; skills should describe repeatable workflows.
